Templates

A template is an XML (or HTML) file that can be bound to some Scala class extending Template. Generically, this binding can be any XML transformation, transforming the source XML to some output XML. A template is an XML transformation itself, so they compose nicely.

The usual way to bind an XML template to code is to use CSS selectors in the bind method.

Here's an example: 
        class LoginScreen extends Template {
          val bind =
            "#title" #> "Welcome!" & 
            "#menu .item a" #> SetAttr("href", "/home")  
        }
