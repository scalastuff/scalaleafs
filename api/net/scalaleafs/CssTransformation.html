CssTransformation

class CssTransformation[A <: (NodeSeq) ⇒ NodeSeq] extends XmlTransformation

A css transformation is a transformation based on a css selector. An instance is typically obtained through the #> operator, e.g. 

"#mydiv" #> SetClass("selected")
